Employee-Data-Analysis Project 

Project Overview
This project focuses on analyzing employee data to derive meaningful business insights related to salary distribution, department-wise cost, and city-wise workforce presence.
The analysis simulates a real-world HR / MIS analytics use case.
Tools Used
- Microsoft Excel (Pivot Tables, SUMIFS, Averageif, Averageifs, COUNTIFS)
- SQL (GROUP BY, SUM, AVG, COUNT)
- Power BI (Dashboard & Visuals)
Dataset
Employee dataset containing:
- Employee ID
- Department
- Salary
- City
- Experience
- Performance Rating
Key Analysis Performed
- Department-wise salary cost analysis
- City-wise employee distribution
- Average salary by department
- Identification of highest cost department and city
Skills Demonstrated
- Data Cleaning
- Data Aggregation
- Business Insight Generation
- Dashboard Thinking
Project Status
Ongoing (Enhancements in progress)
